Cloud Software / SaaS

LedgerSeiri（レジャーセイリ）

帳簿データを、正しく整理する。

LedgerSeiri は、クロスボーダー EC 事業者向けに、
売上・手数料・費用などの会計データを整理・可視化することに特化した クラウド型ソフトウェア（SaaS）です。

複数の EC プラットフォームや決済サービスから取得したデータを統合し、 会計・記帳・分析に利用しやすい形で提供します。

主な機能

  • EC 売上・手数料・費用データの整理
  • 会計・記帳用データの集計および一覧化
  • CSV / Excel 形式でのデータ出力

料金

LedgerSeiri はサブスクリプション型のソフトウェアサービスです。

現在は β版（月額 0 円・税込） を提供しています。 正式な有料プラン開始時には、申込・決済画面にて料金を明示します。
